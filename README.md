# mysql-ecommerce-sales-analysis
SQL project analyzing e-commerce sales data using MySQL. Includes table creation, queries, aggregations, and business insights.

# MySQL E-Commerce Sales Analysis

## Project Description
This project analyzes e-commerce sales data using MySQL.
The goal is to understand sales performance, customers, and product trends.

## Tools Used
- MySQL
- SQL (Joins, Group By, Aggregate Functions)
- GitHub

## Dataset
The dataset contains tables related to:
- Customers
- Orders
- Products
- Sales transactions

## What I Analyzed
- Total sales by product
- Sales by customer
- Monthly sales trends
- Top selling products

## Key Insights
- Identified top revenue-generating products
- Found high-value customers
- Observed sales growth patterns
